Optimal gaps

Pequeño script para obtener los espaciados optimos entre elementos para cuando la ventana cambia de tamaño y que el ultimo elemento rebase el viewport para que la barra de desplazamiento este justificada y tenga sentido

Los espaciados son calculados para que el ultimo elemento este rebasado, ya sea dando pixeles o el porcentaje en la opción llamada "offset"

Los espaciados se encojeran con un calculo linear dese el punto en el que aparece la barra de desplazamiento y hasta el punto de la opcion "minViewportWidth". Este calculo linear es usado en este articulo de tipografia fluida

Como usarlo:

      
        import OptimalGaps from "optimalGaps.min.js";

        const container = document.querySelector('.container');
        const elements = document.querySelectorAll('.element');

        const options = {
          container,
          elements,
          offset: '30%',
          initialGap: 20,
        };

        new OptimalGaps(options);

Parametros

Parametros Requerido Tipos & valores aceptados Default Descripción
container Si [DOMNode] - null El contenedor que tiene la propiedad de CSS overflow
elements Si [NodeList] - null La lista de elementos donde se calcularan los espacios
offset Si [String] '100px' or '10%' null La cantidad del ultimo elemento que se rebasara la vantana
initialGap Yes [Int] - null Los espaciados que tendran sin iniciar el calculo
breakpointStart No [Int] - null Si se pone está opcion, los calculos iniciaran a partir de este punto en el viewport
containerClass No [String] - 'js-has-scroll' The class that will be added to the contianer when the gap is calculated
minimunGap No [Int] - 10 The minimun gap that will be set on the calculations
minViewportWidth No [Int] - 360 The size of the viewport when the calculation will stop and the minimun gap will set
cssCustomProperty No [String] - '--optimal-gap' The CSS custom property that will be set on the container that has the calculated gap
sidePaddings No [Int] - 0 The amount of padding in pixels that will be set in the container element with a CSS custom property: '--side-paddings'
